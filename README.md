# 🚀 Automated Lead Capture & CRM System

## 📌 Overview
This project demonstrates a complete end-to-end automation system that captures leads from a website and processes them through a CRM with automated follow-ups.

## 🌐 Live Demo
https://alimrnobody.github.io/bright-smile-crm-project/

## 🔄 System Workflow
1. User submits form on landing page
2. Data sent to n8n via webhook
3. Contact created in Go High Level (GHL)
4. Automated email & SMS follow-ups triggered
5. Lead data stored in Google Sheets
6. Appointment booking updates CRM & Sheet

## 🛠 Tech Stack
- WordPress (Landing Page)
- Go High Level (CRM & Automation)
- n8n (Workflow Automation)
- Google Sheets (Data Logging)

## 📂 Project Structure
- wordpress-site → Landing page files
- n8n-workflows → Automation JSON files
- ghl → CRM setup & workflows
- google-sheets → Data structure
- docs → Case study

## ⚙️ Key Features
- Automated lead capture
- CRM pipeline management
- Email & SMS automation
- Appointment booking system
- Google Sheets integration

## 💡 Use Case
Designed for a dental clinic to automate lead handling and appointment scheduling.
